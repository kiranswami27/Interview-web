# **App Name**: Interview Insights

## Core Features:

- Animated Landing Page: Landing page with animated introduction and clear call to action to upload the video.
- Processing Screen: Dynamic processing screen showing real-time analysis progress.
- Interactive Video Player: Interactive video player with timestamped issue markers.
- Analysis Dashboard: Analysis panel displaying the speech transcript with highlighted filler words, tone/emotion insights, body posture feedback via frame thumbnails, and clarity metrics.
- Reasoning Analysis Tool: The tool determines relevant feedback points.
- Export to PDF: Allows exporting the analysis as a PDF report.

## Style Guidelines:

- Primary color: Deep Blue (#2c3e50) for a professional and trustworthy feel.
- Background color: Light Gray (#f0f2f5) for a clean and modern interface.
- Accent color: Teal (#3498db) for highlights and interactive elements.
- Secondary color: Warm Gray (#7f8c8d) for less important text and UI elements.
- Error color: Soft Red (#e74c3c) for error messages and warnings.
- Body and headline font: 'Inter' sans-serif for a clean and modern interface suitable for headlines and body text.
- Use sharp, minimalist icons for clear communication of data points.
- Use subtle transitions and loading animations with Framer Motion to enhance user experience.
- Split-view layout for the dashboard, interactive video player on the left and rich analysis panel on the right.
